# 🏆 Campeonato Profissional
Projeto completo para gerenciamento de campeonatos de futebol, desenvolvido em duas versões diferentes.

---

## 📁 Estrutura do Projeto
### 🌐 1 - Versão Web
Interface visual moderna, feita com **HTML, CSS e JavaScript puro**. Funciona direto no navegador, sem necessidade de instalação.

**Funcionalidades:**
✅ Cadastro, edição e exclusão de times
✅ Geração automática de confrontos
✅ Registro manual e simulação de partidas
✅ Cálculo automático de classificação, pontos e saldo de gols
✅ Cadastro de jogadores e ranking de artilheiros
✅ Estatísticas gerais e opção para reiniciar o campeonato do zero

---

## 🐍 2 - Versão Python
Sistema de linha de comando feito em **Python**, para uso no terminal. Permite registrar jogos e consultar a classificação de forma prática.

---

## 🚀 Como usar
- **Versão Web**: basta abrir o arquivo `1 - Versão Web/index.html` no navegador.
- **Versão Python**: execute o arquivo `2 - Versão Python/Campeonato.py` com o Python instalado.
